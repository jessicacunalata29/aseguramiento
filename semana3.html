📚 Factores de Calidad de Software

Resumen

Los factores de calidad son características que permiten medir qué tan bueno es un software. Según la norma ISO/IEC 25010, ayudan a garantizar que un sistema funcione correctamente, sea fácil de usar y pueda crecer sin perder rendimiento.

Factores más importantes

  • Funcionalidad: el sistema debe realizar correctamente las tareas para las que fue creado.
  • Confiabilidad: el sistema debe funcionar de manera estable y con pocos errores.
  • Usabilidad: debe ser fácil de aprender y utilizar.
  • Eficiencia: debe aprovechar correctamente los recursos y responder rápidamente.
  • Escalabilidad: debe soportar el crecimiento de usuarios y datos.
  • Portabilidad: debe poder ejecutarse en distintos dispositivos y sistemas operativos.

💡 Caso Práctico

Sistema de Matrículas del Instituto

  • Funcionalidad (90%): registra estudiantes, materias y horarios correctamente.
  • Confiabilidad (80%): presenta pocas fallas durante el proceso de matrícula.
  • Usabilidad (85%): la interfaz es sencilla y fácil de aprender.
  • Eficiencia (75%): el tiempo de respuesta es de 3 segundos.
  • Escalabilidad (70%): soporta hasta 500 usuarios simultáneos.
  • Portabilidad (95%): funciona en computadoras, tabletas y teléfonos móviles.

Conclusión

Los factores de calidad permiten identificar las fortalezas y debilidades de un sistema para mejorarlo continuamente y ofrecer una mejor experiencia a los usuarios.
